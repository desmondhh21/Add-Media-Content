Adding Media Content

Tags & Uses

Name Tag Use
Anchor <a> The anchor element is used to create hyperlinks in an HTML document. The hyperlinks can point to other webpages, files on the same server, a location on the same page, or any other URL via the hyperlink reference attribute, href. It is important to add <target="_blank"> after your link so that it prompts the brower to open a new window that directs to the intened webpage.
Image <img> The <img> HTML element embeds an image into the document. The attribute <src> is required.
Audio <audio> This element is used to embed sound content in documents. It may contain one or more audio sources, represented using the <src> attribute or the <source> element: the browser will choose the most suitable one. The audio element does not need a closing tag.
Video <video> The video element embeds a media player for video playback. The content inside the opening and closing tag is shown as a fallback in browsers that donâ€™t support the element. Ex. "Video Not Supported"
Embed <embed> This element embeds external content at the specified point in the document. This content is provided by an external application or other source of interactive content such as a browser plug-in.
